# Luvo MiniApp Backend

Приложение использует данные из [Google таблицы](https://docs.google.com/spreadsheets/d/1uaCAzQPy2yVY3tEU3awT9sLI8t7giSMA-ThxS5arSBA/edit?usp=sharing),
где хранится информация о пунктах выдачи, товарах и категориях.

## Health check и прогрев

В приложении есть лёгкий эндпоинт `/health`, который не требует авторизации и
возвращает `{ "status": "ok" }`.

Для поддержания приложения «тёплым» можно настроить периодический запрос, например через cron:

```
*/5 * * * * curl -fsS https://<ваш-домен>/health > /dev/null
```

Время обработки каждого запроса логируется, что позволяет отслеживать стабильность отклика.
